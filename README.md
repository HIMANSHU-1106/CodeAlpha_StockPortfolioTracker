# CodeAlpha Stock Portfolio Tracker

## Description
A simple Python program that calculates the total investment value of a stock portfolio using predefined stock prices.

## Features
- User enters stock names and quantities
- Uses predefined stock prices
- Calculates total investment value
- Handles invalid stock names

## Concepts Used
- Dictionary
- Loops
- Input/Output
- Basic Arithmetic

## Available Stocks
- AAPL = $180
- TSLA = $250
- GOOGL = $150

## How to Run

1. Download the project.
2. Open terminal in the project folder.
3. Run:

python stock_tracker.py
